## DEVOPS
> É a criação de uma rotina de automação para que seja possibilitado a entrega continua de tarefas. Utilizando a junção de DevOps Desenvolvimento e operação unidos, mais ligado com o lado cultural da empresa.

## SRE
> Para manter a rotina de task e controle da equipe, é designada uma pessoa propria para fazer esse gerenciamento. Pode ser explicada como um sistema de regras, inicialmente criado pelo google e é gerenciado por certas normas especificas.


>Ele permite adicionar tarefas, marcar como concluídas e usar um timer para acompanhar o tempo de foco.
